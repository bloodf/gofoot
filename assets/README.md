# GoFoot — Brand & Audio Assets

> This folder contains the pre-generated visual and audio assets shipped with the v4 master prompt.
> The game is open-source (MIT). Brand assets are original. SFX + TTS samples are generated by MiniMax audio models and are free to use inside GoFoot.

|---|---|---|---|
| `logo.svg` | SVG | 1000×1000 | Primary emblem. Square. Use in: top-bar, app icon source, social avatars, favicon (scale up). |

### How the Codex prompt should consume this

`scripts/fetch-assets.ts` is **already complete** for visual assets — the three SVGs here are the canonical sources. The icon, splash, and PWA `manifest.webmanifest` icons can be generated from `logo.svg` (PWA needs 192, 256, 384, 512 + maskable). The sub-agent `asset-fetcher` should:

1. Copy `logo.svg` → `public/brand/logo.svg`.
2. Copy `favicon.svg` → `public/favicon.svg` AND `public/icons/icon.svg`.
3. Rasterize `logo.svg` to PNG sizes 192, 256, 384, 512, maskable-512.
4. Update `nuxt.config.ts` `pwa.manifest.icons[]` with the generated PNGs.
5. Update `nuxt.config.ts` `app.head.link[rel=icon]` to `/favicon.svg`.

| File | Used as |
|---|---|

| `tts-sample-halftime-pt.mp3` | Sample half-time narration (pt-BR). |
| `tts-sample-fulltime-pt.mp3` | Sample full-time narration (pt-BR). |

| `tts-sample-goal-en.mp3` | Sample narration in EN. |
| `tts-sample-halftime-en.mp3` | Sample half-time narration (EN). |

The game uses **`window.speechSynthesis` (Web Speech API)** at runtime, not these MP3s. The MP3s are previews to help the user approve the voice quality before building.

## Build-time integration

`scripts/fetch-assets.ts` (to be written by the `asset-fetcher` sub-agent):

```ts
// pseudo
import { readFileSync, writeFileSync } from 'node:fs'
import sharp from 'sharp'

const LOGO = readFileSync('assets/logo.svg')
for (const size of [192, 256, 384, 512, 1024]) {
  await sharp(Buffer.from(LOGO)).resize(size, size).png().toFile(`public/icons/icon-${size}.png`)
}
writeFileSync('public/favicon.svg', LOGO)
```

Audio files are **bundled** in `public/audio/sfx/` and lazy-loaded by `useSfx()`.
